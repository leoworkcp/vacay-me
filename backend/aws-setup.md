## Packages to install in your backend

### multer

- npm install --save multer

### aws-sdk

- In the Browser
add the following script tag to your HTML pages:
<script src="https://sdk.amazonaws.com/js/aws-sdk-2.849.0.min.js"></script>

- In Node.js
  npm install aws-sdk

- In React Native
  npm install aws-sdk

### body-parser

  API
